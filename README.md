# 💅 Agendamento Manicure - Backend

Sistema de gestão de agendamentos desenvolvido para automatizar o controle de horários de atendimento. Este projeto faz parte da minha transição de carreira para **Desenvolvedor Backend**.

## 🚀 Tecnologias Utilizadas
* **Node.js**: Ambiente de execução.
* **Express**: Framework para construção da API REST.
* **PostgreSQL**: Banco de dados relacional para persistência dos agendamentos.
* **Bcrypt**: Criptografia de senhas para segurança dos usuários.
* **Dotenv**: Gerenciamento de variáveis de ambiente e segurança de credenciais.

## 🛡️ Boas Práticas Implementadas
* **Segurança**: Uso de `.gitignore` e `.env` para proteção de dados sensíveis.
* **Arquitetura**: Organização de rotas e conexão com banco de dados de forma modular.
* **Versionamento**: Histórico de commits organizado seguindo padrões de mercado.

## ⚙️ Como rodar o projeto
1. Clone o repositório.
2. Instale as dependências com `npm install`.
3. Configure o arquivo `.env` com suas credenciais do banco (veja `.env.example`).
4. Execute com `npm run dev`.
